Claude Timer (AI Usage Tracker)

Claudeのメッセージ制限リセットまでの残り時間を、ブラウザのツールバーにバッジ表示するChrome拡張機能です。

インストール方法

  1. ZIPファイルをダウンロードして解凍する
  2. Chrome で chrome://extensions/ を開く
  3. 右上の「デベロッパーモード」を ON にする
  4. 「パッケージ化されていない拡張機能を読み込む」をクリック
  5. 解凍したフォルダを選択

使い方

  1. ブラウザ右上のアイコンをピン留めする
  2. アイコンをクリック → Claude の使用量ページが開く
  3. 自動的にリセット時間を読み取り、バッジに表示
  4. リセット完了で緑色の「OK」表示

注意事項

  • Claudeの画面デザインが変わると読み取れなくなる可能性があります
  • 非公式の学習用プロジェクトです
